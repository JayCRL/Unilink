# Unilink - 现代化云端同步管理系统

Unilink 是一个基于 Go 语言开发的轻量级云端文件同步方案，包含**高性能服务端**与**高颜值 CLI 客户端**。

## ✨ 核心功能

### 1. 智能连接 (Smart Dual-Link)
- **自动探测**：启动时自动在公网 (8.162.1.176) 与局域网 (192.168.1.119) 之间寻找最佳链路。
- **无感切换**：公网不通时 3 秒内自动降级至局域网，确保服务永不离线。
- **手动控制**：支持 `switch` 指令随时手动跳转链路。

### 2. 极致 CLI 交互
- **美学设计**：采用温润的米棕色 (Beige) 配色方案，视觉体验优于传统终端。
- **全汉化反馈**：所有的操作提示、错误报警、成功反馈均已完成深度汉化。
- **增强型 ls**：同时展示云端文件 (☁) 与本地文件 (📁/📄)，并附带目录切换功能。

### 3. 后端管理系统
- **Admin 面板**：内置 HTML5 管理后台 (`/admin`)，采用全中文界面。
- **配额管理**：实时调整每个用户的存储上限 (MB)。
- **安全加固**：数据库连接预检机制，确保服务启动即稳健。

## 🚀 快速开始

### 客户端 (Windows)
1. 进入 `unilink-go` 目录。
2. 运行 `unilink.exe <用户名>`。
3. 输入访问密钥（输入过程完全隐藏，保护隐私）。
4. **常用指令**：
   - `ls`：查看云端与本地文件。
   - `cd <路径>`：切换本地电脑目录。
   - `upload <文件名>`：上传文件。
   - `download <文件名>`：下载文件。
   - `switch`：切换公网/内网模式。

### 服务端部署 (Ubuntu)
1. 将 `unilink-server-linux` 上传至服务器。
2. 执行安装脚本：`sudo ./deploy_ubuntu.sh`。
3. 访问 `http://你的IP:7890/admin` 进行用户管理（默认 Token: `unilink_admin_2026`）。

## 🛠 技术栈
- **Language**: Go 1.20+ (Backend & Client)
- **Database**: MySQL (支持 Docker 部署)
- **UI**: Embedded HTML5 / CSS3 (Admin Panel)
- **Security**: x/term (Secure Input), Admin-Token Auth
